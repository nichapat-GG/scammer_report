# scammer_report
Project of Computer Programming 

โปรแกรม Scammer report เป็นโปรแกรมสำหรับรายงานและเก็บข้อมูลเบอร์โทรศัพท์ที่มีความเสี่ยงว่าจะเป็นมิจฉาชีพ โดยมี 3 ฟังก์ชัน ดังนี้

1. การเก็บรวบรวมข้อมูลจากผู้ใช้งาน :
   โดยจะมีช่องสำหรับการแจ้งข้อมูลว่า "เบอร์นี้เป็นเบอร์ที่น่าสงสัยว่าเป็นมิจฉาชีพ" โดยจะเก็บรวบรวมข้อมูลพวกนี้ไว้สำหรับฟังก์ชันต่อไป
   
2. การค้นหาว่าเบอร์นี้เป็นมิจฉาชีพหรือไม่ หรือเคยมีการแจ้งเข้ามาไหม :
   เมื่อมีการค้นหาแล้วพบว่าข้อมูลตรงกัน โปรแกรมจะปรากฎข้อมูลว่าเบอร์นี้ใช่หรือไม่ใช่มิจฉาชีพ พร้อมบอกความเสี่ยงของเบอร์นั้นว่ามีมากน้อยแค่ไหน แล้วถ้าหากไม่มีข้อมูลว่าเบอร์นี้เป็นมิจฉาชีพ จะมีการถามต่อว่าอยากจะแจ้งว่าเบอร์นี้เป็นเบอร์ของมิจฉาชีพหรือไม่ ดำเนินต่อเป็นฟังก์ชันต่อไป
   
3. การแสดงเปอร์เซ็นต์ความเสี่ยง
   เป็นสิ่งที่จะปรากฎพร้อมกับการระบุว่าเบอร์นี้เป็นเบอร์ของมิจฉาชีพหรือไม่ โดยจะคำนวณจากการแจ้งเข้ามาว่าเบอร์นี้มีการแจ้งไปกี่ครั้งแล้ว ซึ่งถ้ามีการแจ้งเข้ามามาก เบอร์นี้ก็จะมีความเสี่ยงสูง
